# Jua-JS
Jua.js is a client side java script works offline. You can do anything with html nodes and some awesome inbuild Functions.


``` javascript
Jua("#div")
```
By This You Can Select Elements and It Will Do Magic 

``` javascript
Jua("#div").css()
Jua("#div").class()
Jua("#div").attr()
Jua("#div").anime()
Jua("#div").anim()
Jua("#div").animate()
```
